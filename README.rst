A package to ease running some MD and DFT codes, particularly on the Imperial HPC.

Installation
------------

As outlined at: https://wiki.imperial.ac.uk/display/HPC/Python

::

    module load anaconda3/personal

    anaconda-setup

    pip install atomic-hpc
